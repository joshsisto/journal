## Suggestion 32: Use `current_app.config.get()` for `common_passwords` in `register_user()` and `change_user_password()`

**Before (in `register_user()` and `change_user_password()`):**
```python
        common_passwords = ['password', '123456', 'qwerty', 'admin', 'welcome']
        if password.lower() in common_passwords:
            return None, 'Please choose a stronger password.'
```

**After (in `register_user()` and `change_user_password()`):**
```python
        if password.lower() in current_app.config.get('COMMON_PASSWORDS', []):
            return None, 'Please choose a stronger password.'
```
